Nuxeo Digital Asset Management 1.1 Release Notes

Welcome to Nuxeo Digital Asset Management (DAM) 1.1.

Download latest release of Nuxeo DAM from http://www.nuxeo.com/downloads/

Requirements

Nuxeo DAM 1.1 needs Java. Please set up recommended Sun Java Runtime Environment (JRE) 1.5 if it's not already installed on your computer. Nuxeo is also compliant with Java 6 (see specific notes about Java 6). It won't work with any previous release of Java (e.g. Java 1.4). Under windows, virus scan should be configured to avoid scanning Nuxeo's intallation folder. More details on this FAQ entry .

Getting Started

After you have installed Nuxeo DAM:

Either follow the Quick Start Guide or the instructions below.

  1. Go to your installation directory

  2. Start the application

    • under Linux/Unix/MacOS, run "./bin/nuxeoctl start" (run "./bin/nuxeoctl help" for list of available commands).

    • under Windows, run "bin\nuxeoctl.bat".

  3. Point your browser to http://localhost:8080/nuxeo/

  4. Log in using the "Administrator" login and the "Administrator" password (no quotes, of course, but capital "A").

Edit nuxeo.conf if you want to change default values. By default, JBoss is binded only on all available addresses (BINDHOST=0.0.0.0), change the BINDHOST value if you want to set a specific IP (recommended).

Complementary features

The following features depend on the environment where you installed Nuxeo DAM.

  • PDF transformation: this feature requires OpenOffice started as server, complete the installation by following OpenOffice chapter in Nuxeo-Book for Administrators.

  • Preview and annotations: requires install of third-party software "pdftohtml",see "Preview and Annotations - Pdf documents" chapter.

  • Image tiling: requires install of third-party software "ImageMagick", see "Preview and Annotations - Image tiling" chapter.

  • Video ffmpeg from http://ffmpeg.arrozcru.org/autobuilds/ provided by this website should be decompressed with: http://www.7-zip.org/. Extract the ffmpeg archive into a new folder named C:\ffmpeg for instance.

  • Video Streaming (Windows only)

About Nuxeo DAM

Nuxeo DAM is a robust, extensible, Digital Asset Management solution available as Open Source Software (OSS) that helps marketing and brand managers catalogue, process, store, retrieve, and distribute rich media content, including images, audio, and video.

Streamlining these tasks enables creative users to focus on ideas and design by freeing them from resource-intensive content controltasks.

Nuxeo DAM, a state-of-the-art application built on a new generation of ECM, provides:

* An intuitive experience, designed for creative users

* Improved productivity, with the automation and streamlining of routine tasks

* A trusted repository, designed for secure and easy scaling as projects expand

* Adaptability, with Nuxeo Studio, a configuration and customization environment

* Interoperability, with a CMIS-enabled platform

Nuxeo DAM is a packaged application based on Nuxeo EP (Enterprise Platform), an extensible, Java-based ECM platform. Nuxeo EP is CMIS-enabled, thus ensuring that Nuxeo DAM benefits from the interoperability capabilities of the underlying plat- form. What does this mean? Organizations can safely invest in any number of CMIS-compliant Web Content Management offerings, and be assured that their rich media assets in Nuxeo DAM are easily distributed to create an engaging web experience.

Nuxeo DAM new features:

  • Bulk Import

    With Nuxeo DAM you can import for large amount of files, with its metadata

  • Video Story-boarding

  • Video Streaming

    Streaming Feature offers to user to share through Nuxeo DAM their video (and soon audio) in streaming, that mean user don't need to wait for the end of the download of the file to start to see it. Most used video format can uploaded into Nuxeo DAM. Available for Windows servers.

  • Bulk Edit

    Nuxeo DAM allows user to edit a selection of assets.

  • Nuxeo DAM and Nuxeo DM on the same repository

    Nuxeo DAM and Nuxeo DM can reach the same database. That way, documents can be created, enriched, go through workflows, be approved, and then, published in the asset library, to be visible in the DAM interface.

This release note presents the main new features of the platform, through a default web client.
