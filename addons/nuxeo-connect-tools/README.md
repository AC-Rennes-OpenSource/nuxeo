# Nuxeo Connect Tools

Reports configuration and metrics of running nuxeo instances for being attached 
to nuxeo JIRA's ticket.

# Building

    mvn clean install

# Installing

Install the jar in the nuxeo's bundles folder.

# Running

Invoke  a MBean
