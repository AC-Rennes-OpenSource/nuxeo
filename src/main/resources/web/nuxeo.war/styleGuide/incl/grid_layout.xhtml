Grid Layout

The Grid layout is here to help you align blocks and widgets into columns and rows in a specific area. By following a pattern made of containers with CSS classes, you will easily design your customized tab content, keeping a visual consistency, readablity, and adaptability between screens.

Containers construction

Your layout must contain the following elements:

  • A div .gridContainer, containing several div .gridRow vertically.
  • Each .gridRow contains several .gridBox horizontally.
  • A div .gridBox may contain several .widgetPanel vertically.
  • Each div .gridBox got a secundary .gridStyle** class to define its width.

The .gridStyle** classes represent the width of each .gridBox container. The sum of all the .gridBox in a .gridRow line must always be equal to 12.

A 12-column based grid

The grid model included in the Nuxeo Platform styles is 12-column based. It means that each .gridRow must be equal to 12, regarding the numbers of the .gridStyle** class. Example of which combination you can write:

  • .gridStyle12 = a 100% width layout
  • .gridStyle6 + .gridStyle6 = a 50% / 50% layout
  • .gridStyle8 + .gridStyle4 = a 66% / 33% layout
  • .gridStyle4 + .gridStyle4 + .gridStyle4
  • .gridStyle6 + .gridStyle3 + .gridStyle3

.gridBox .gridStyle12

Containers specifications

The .gridContainer

It is the main container of all your layout. No margins and padding, it is only a frame that will encompass your rows.

The .gridRow

No padding and margins neither, the .gridRow will shelter each block you want to add. At the end of all the gridRow, don't forget to add a div class="clear" closing tag to stop the .gridBox imbrication.

The .gridBox

This class only define the floating behavior of each box, allowing them to move down if the space is missing.

The .gridStyle**

The secundary class to add to each .gridBox to define the width.

Example: Summary View

The best example of the grid usage is the summary view of a document. You can design your layouts from the Tab feature of Nuxeo Studio.

This layout contains one .gridRow in the .gridContainer, made of two .gridBox divs: .gridStyle7 + .gridStyle5. The div .gridStyle7 contains one widget for the picture, the div .gridStyle5 contains 4 widgets.
