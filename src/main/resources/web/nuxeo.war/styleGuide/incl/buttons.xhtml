Buttons

The styles of buttons can be applied on button or links. For each forms in default pages, search, modal pop ups, here is a showcase of which classes you can apply on your actions.

Available classes

Link class button

The default style for buttons and links is handled by the css class .button. For small area, you can add the complementary class .smallButton to reduce font-size and space.

When an action in a form is disabled, or only abled in a particular case, use the .disabled class in addition. The .disabled="disabled" attribute needs to be used for a right behavior.

If you need to highlight an action among several buttons, you can use the .major class in addition. It will guide the user to the most logical action.

Buttons containers

The available actions displayed just under the tabs are displayed in a div class .action_bar. Then, each action is listed in a .action_bar in it.

If you need an icon in your button to help user to quickly identify the action, use an img tag first and then display your label in your link. The class .button is not mandatory, use it here will only change the margin.
