Boxes

Sometimes your content needs to be gathered in a box: listings, small forms, options, filters. The platform contains two types of boxes: foldables boxes with options, and bubble boxes. For each model, chose a title, define your content, select a model and think about the place you want to display it: the container of your box will handle the global width of your box.

Bubble boxes

A div class .bubbleBox contains a first div class .bubbleHeader with a h3 title, and then a div class .boxBody for the main content.

Simple bubble box

Borders, margin, a place for a title, and a content in a div class .bubbleBox.

Very simple.

Box with button

Add an input class .button
before the h3 in the bubbleHeader.

Box with shade

Add a secundary .bubbleShadow
to the default .bubbleBox class

Default foldable boxes

A div class .foldableBox contains a h3 title class .unfolded or .folded with a link in it, and then a div class .boxBody for the main content.

.unfolded h3 title

The foldable box content in a div .boxBody. The h3 title class switch from .folded to .unfolded and vice versa.

Foldable box with button Enable

You can include a tooltip in the title by using an img class .layoutHelpLabel. Write the content of your helper in the title of your img tag. You can also add an action, by adding a link class .button smallButton action.

Custom foldable boxes

.foldedDisabled h3 title

If you want to use the foldable box style only, without the foldable behavior, remove the link from the .h3 and add a .foldedDisabled to it.
Foldable box div title class .unfolded

A light foldableBox can also be an option for light interface: use the class .foldableLightBox for the container, then keep the same layout than the default foldableBox.
